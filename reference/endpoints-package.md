# endpoints: Simulate Correlated Multi-Endpoint Clinical Trial Data

Simulate correlated multi-endpoint clinical trial data using a Gaussian
copula approach. This package provides tools to generate realistic
patient-level datasets with configurable treatment effects and
covariance structures. It supports multiple endpoint types (continuous,
binary, count, and time-to-event) across multiple treatment arms.
